# Sistema de Gestión Operativa - Compañía de Bomberos Chosica N° 32

1. Abra esta carpeta en Visual Studio Code.
2. Abra la terminal integrada y ejecute una sola vez:

```powershell
pip install -r requirements.txt
```

3. Inicie la aplicación:

```powershell
python simulacion_sistema_gestion_operativa.py
```

4. Abra en el navegador: `http://127.0.0.1:5000`

Regístrese con código de 6 caracteres, nombres, apellidos y contraseña de 7 caracteres. El registro inicia la sesión automáticamente. Desde el panel podrá registrar entrada/salida, asignar guardias y cerrar sesión.
